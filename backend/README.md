# Backend

Express API for authentication, video analysis, SEO idea discovery, AI hook generation, and billing checkout.

## Routes

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/auth/me`
- `POST /api/analyze`
- `GET /api/seo?niche=fitness`
- `POST /api/ai/hooks`
- `POST /api/payments/checkout`

## Local run

```bash
npm install
npm run dev
```
