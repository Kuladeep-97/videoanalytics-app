# API Routes

## Authentication

- `POST /api/auth/register`
- `POST /api/auth/login`
- `GET /api/auth/me`

## Video Analyzer

- `POST /api/analyze`

## SEO Finder

- `GET /api/seo?niche=<value>`

## AI Content

- `POST /api/ai/hooks`

## Billing

- `POST /api/payments/checkout`
- `POST /api/payments/confirm`
